Code editor

Amount of use:

Extensive. Used for every homework.

Expected knowledge:

Extensive

Coverage in class:

None. You should be able to use a text based code editor well. Examples include Notepad++, Sublime Text, Emac, Vim or TextWrangler

My recommendation: Sublime

FTP Program

Amount of use:

None. You should be able to upload files to your engr directory already. You should also know how to set permissions on those files.
